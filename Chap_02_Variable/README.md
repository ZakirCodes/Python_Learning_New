# 📌 Chapter 02 - Variables

## 📖 Concept Overview
This chapter covers one of the most fundamental concepts in Python — **Variables**. A variable is a name given to a memory location that stores a value, which can be a number, text, or any other type of data.

```python
name = "Zakir"
print(name)
```

Variables let a program store, reuse, and update data instead of writing the same value again and again — making them the backbone of almost every Python program.

---

## 🎯 What I Learned
- Creating variables to store integers, decimals (float), and strings
- Assigning one variable's value to another variable
- Performing arithmetic operations directly on variables (`x + y`, `x + 2`)
- Declaring multiple variables in a single line (`p, q, r = 2, 5, 7`)
- Different variable naming conventions: Pascal Case, Camel Case, Flat Case, and Snake Case
