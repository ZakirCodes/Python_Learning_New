# 📌 Chapter 01 - Print()

## 📖 Concept Overview
This chapter covers the very first building block of Python programming — the `print()` function. It is used to display output (text, numbers, variables, or results) on the screen.

```python
print("Hello, World!")
```

`print()` is often the first function every programmer learns, and it's used throughout every Python program for debugging, displaying results, and communicating with the user.

---

## 🎯 What I Learned
- How to use the `print()` function to display text and values
- Printing multiple values in a single `print()` statement
- Using `sep` and `end` parameters to control output formatting
- Printing strings, numbers, and combinations of both
- Basic string formatting inside `print()`
